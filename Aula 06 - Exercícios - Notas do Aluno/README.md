## _Programa de Aprovação Escolar_

### _Etapas_

>Defina como constante a quantidade de notas de um aluno;
>
>Capture nome e cada uma das notas;
>
>Calcule a média.
>

### _Informações_
>
> Reprovou (<3)
>
> Recuperação (<7)
>
> Nota mínima (=7)
>
> Nota média (<10)
>
> Nota máxima (=10)
>
> Digitação incorreta informe "ERROR".
