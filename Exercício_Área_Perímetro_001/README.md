# EXERCÍCIO ÁREA E PERÍMETRO

## Explicação do Exercício

- 1° Copiar o enunciado do exercío;

- 2° Definir minhas variaveis (sempre lembrar de colocar "nomeclaturas" explicativas);

- 3° Criar a classe "Scanner" (serve para colocar informações no console);

- 4° Definir as perguntas para respostas do console. (elabore perguntas objetivas);

- 5° Elaborar o código "Aritmético" de acordo com o que o enunciado pede;

- 6° Escrever minhas resposta usando minhas "variaveis" da equação;
