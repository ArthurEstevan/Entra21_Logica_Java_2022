## _Operações Aritméticas_

- (+) operador de adição.

- (-) operador subtração.

- (*) operador de multiplicação.

- (/) operador de divisão.

- (%) operador de módulo ou resto da divisão.

#### _Como escrever o código:_

```
float numeroA=10, numeroB=10;

System.out.println("Realizando Operações " + (numeroA + numeroB) );
System.out.println("Realizando Operações " + (numeroA - numeroB) );
System.out.println("Realizando Operações " + (numeroA * numeroB) );
System.out.println("Realizando Operações " + (numeroA / numeroB) );
```
