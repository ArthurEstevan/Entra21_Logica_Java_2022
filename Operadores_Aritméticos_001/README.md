# OPERADOR ARITMÉTICO

## Tipos de Operação

- (+) operador de adição.

- (-) operador subtração.

- (*) operador de multiplicação.

- (/) operador de divisão.

- (%) operador de módulo ou resto da divisão.

 Note que no projeto ainda continuamos usando variaveis, comentarios e constantes. Apenas aprendemos mais um modo de incrementar.
